# Chroma 56

Wong et al. (2025) の成人絶対音感訓練を参考にした、依存関係なしのWebアプリです。1音版と2音版をトップページから選べます。

## オンラインで使う

https://toshiotawa.github.io/chroma-56/

## 起動

`index.html` をブラウザで直接開くか、このフォルダでローカルサーバーを起動します。

```sh
python3 -m http.server 4173
```

その後、`http://localhost:4173` を開いてください。

## 構成

### 1音バージョン

- 56日 × 160問 = 8,960問
- Fから始め、E、F#、D#、G、D、G#、C#、A、C、A#、Bの順に追加
- 各音を「記憶・境界・速度・定着」の4日で学習
- 最終8日は全12音の統合、音色・音域汎化、高速判断、最終確認
- OOB比率は記憶25%・境界50%・速度30%・定着25%。境界音は半音隣を2半音隣より厚く出題
- 3音域×5音色のセルを均等に組んだ1日分のデッキをシャッフルして出題
- Sine、Triangle、Sawtooth、Square、FMの5音色を使用
- 無フィードバック確認前に20秒のShepard toneを再生

### 2音バージョン

- 56日間、E・F・A・Bbの4音だけを使用
- 毎問必ず2音。単音、OTHER、別のピッチクラスは混ぜない
- 中央の各ペア → 共通音ブロック → 同一オクターブ → 片側移動 → ペア別3×3 → 全54配置の順で細分化
- 各日の出題プールを巡回してから反復するため、ランダムの偏りで配置が抜けない
- 3オクターブ（3・4・5）と4音色を使用
- 正答率による合格、日付ロック、DBへの進捗保存はなし

## プライバシーと保存

DB、通信API、Cookie、`localStorage`を使用しません。終了画面から、その回の結果だけを任意でJSONファイルとして書き出せます。

## 注意

元研究は8週間で平均21.4時間・15,327試行でした。このアプリは研究の訓練原理を軽量化した8,960試行版であり、研究プロトコルの完全な複製や絶対音感の獲得を保証するものではありません。聴覚保護のため、小さめの音量から使用してください。
